### 冒泡排序

时间复杂度为O(N^2)，内层循环每次结束的时候会使得“最大”的那个数排到最后。下面的代码是[冒泡排序三种实现](http://blog.csdn.net/morewindows/article/details/6657829)中的第三种，比一般的实现优化了不少。

```//冒泡排序3  
void BubbleSort3(int a[], int n)  
{  
    int j, k;  
    int flag;  
      
    flag = n;  //已完成排序的（从右到左）
    while (flag > 0)  
    {  
        k = flag;  
        flag = 0;  
        for (j = 1; j < k; j++)  
            if (a[j - 1] > a[j])  
            {  
                Swap(a[j - 1], a[j]);  
                flag = j;  
            }  
    }  
}  
```
